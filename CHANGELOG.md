## 0.1.0

* Initial release
* Image rendering: JPG, PNG, GIF, WebP, BMP, SVG, HEIC
* PDF rendering via native platform channels (Android PdfRenderer / iOS PDFKit)
* Video thumbnail + metadata via platform channels
* Audio player UI with animated waveform
* Pure Dart Markdown renderer
* Pure Dart JSON tree viewer with expand/collapse
* Pure Dart CSV table renderer
* Pure Dart ZIP archive browser
* Pure Dart syntax tokenizer (Dart, Python, JS, Kotlin, Java)
* Fallback renderer with file metadata for unknown types
* Zero external package dependencies
